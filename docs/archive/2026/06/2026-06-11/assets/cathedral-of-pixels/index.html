The Pipeline — 像素朝圣之路

Every vertex embarks on a sacred journey through seven stages before becoming a pixel. This renderer implements each one in pure Python — no libraries, no shortcuts.

📐
Linear Algebra
Vec3, Vec4, Mat4 — handwritten with column-major storage
🎯
Transform Pipeline
Model→World→View→Clip→NDC→Screen in sequence
✂️
Back-face Culling
Screen-space area sign test to discard hidden faces
🔺
Edge Walking
Barycentric rasterizer with sub-pixel precision
📏
Z-Buffer
Per-pixel depth testing for correct occlusion
💡
Phong Lighting
Ambient + Diffuse + Specular with per-vertex normals
🎨
Flat & Gouraud Shading
Barycentric color interpolation across triangle faces
📷
Orbit Camera
Azimuth/elevation/distance controls with smooth movement

Why 1993?

1993 was the year Doom shipped — the game that proved software rendering could create immersive 3D worlds on consumer hardware. The techniques it pioneered — BSP trees, affine texture mapping, column-based rendering — shaped an entire generation of graphics programming. This renderer pays homage to that era by building every stage from first principles, proving that the fundamentals remain beautiful even when you can see every multiply.
